# Comandos de Palworld

Palworld incluye comandos administrativos que permiten **gestionar jugadores, el servidor y el mundo**.  
Estos comandos pueden ejecutarse **dentro del juego** o **desde la consola del servidor**.

---

## Acceder a los comandos

### Dentro del juego

1. Presiona `Enter` para abrir el chat
2. Escribe el comando
3. Presiona `Enter` para ejecutarlo

### Desde la consola del servidor

Los comandos pueden ejecutarse directamente desde la **consola del panel** sin estar dentro del juego.

## Comandos de administración

| Comando | Descripción |
|-------|------------|
| `/AdminPassword <password>` | Inicia sesión como administrador |
| `/Shutdown <segundos>` | Apaga el servidor tras el tiempo indicado |
| `/Save` | Guarda el mundo manualmente |
| `/DoExit` | Cierra el servidor inmediatamente |

## Comandos de jugadores

| Comando | Descripción |
|-------|------------|
| `/KickPlayer <SteamID>` | Expulsa a un jugador |
| `/BanPlayer <SteamID>` | Banea a un jugador |
| `/UnBanPlayer <SteamID>` | Desbanea a un jugador |
| `/ShowPlayers` | Muestra jugadores conectados |

## Comandos de servidor

| Comando | Descripción |
|-------|------------|
| `/Info` | Muestra información del servidor |
| `/Broadcast <mensaje>` | Envía un mensaje global |
| `/SetTime <hora>` | Cambia la hora del mundo |

## Comandos de depuración

| Comando | Descripción |
|-------|------------|
| `/LogLevel <nivel>` | Cambia el nivel de logs |
| `/DumpServerInfo` | Muestra información detallada |
