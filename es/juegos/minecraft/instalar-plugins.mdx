# Cómo Instalar Plugins en tu Servidor de Minecraft

Instala plugins para añadir nuevas funcionalidades en tu servidor.

## Instalación Automática

1. Deten tu servidor y saber la versión de tu servidor.
2. Para saber la versión tenemos que ir a **Files > logs > latest.log**

<img src="/images/games/minecraft/plugin-version.png" alt="Plugin" />

3. Una vez que ya sabemos la versión vamos a la sección **Plugins**
4. Buscamos el plugin y verificamos que sea compatible con nuestra versión.
5. En este caso buscaremos el plugin **"World Edit"**
6. Seleccionamos la opción de versions.

<img src="/images/games/minecraft/plugin-version-2.png" alt="Plugin 2" />

7. En este apartado filtramos por nuestra versión (1.21.1) y tipo de servidor (PaperMC)
8. Instalamos la versión más reciente con el boton de **Install**

<img src="/images/games/minecraft/plugin-version-3.png" alt="Plugin 3" />

9. Una vez instalado nos saldrá un mensaje de que se ha instalado correctamente y ya esta listo para ser usado.

<img src="/images/games/minecraft/plugin-version-4.png" alt="Plugin 4" />
