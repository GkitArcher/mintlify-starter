# Cómo unirse a un servidor Project Zomboid

## Requisitos previos

Antes de conectarte a un servidor de Project Zomboid, asegúrate de cumplir con lo siguiente:

- Tener **Project Zomboid** instalado desde Steam  
- Usar una **versión compatible** con el servidor  
- Contar con conexión a internet estable  
- (Opcional) Tener los **mods requeridos** por el servidor  

## Encontrar la IP y el puerto del servidor

Para unirte a tu servidor alojado en **RDSNode**, primero debes obtener su IP y puerto:

1. **Inicia sesión** en el [panel de control de RDSNode](https://panel.rdsnode.com/).
2. **Selecciona tu servidor** de Project Zomboid.
3. Dirígete a la sección de **Información del servidor** o **Terminal**.
4. Copia la **IP pública** y el **puerto** asignado.

## Conectarse al servidor desde el juego

1. Abre **Project Zomboid** desde Steam.
2. En el menú principal, haz clic en **Join** (Unirse).
3. Ve a la pestaña **Internet**.
4. Haz clic en **Add Server**.
5. Completa los campos:
   - **Server Name**: Nombre identificador (puede ser cualquiera)
   - **IP Address**: IP de tu servidor
   - **Port**: Puerto del servidor
   - **Username / Password**: Credenciales del jugador
6. Guarda el servidor y haz doble clic para conectarte.

## Conexión mediante Steam (opcional)

También puedes unirte usando Steam:

1. Abre **Steam**.
2. Ve a **Ver → Servidores**.
3. En la pestaña **Favoritos**, haz clic en **Agregar servidor**.
4. Ingresa la IP y puerto en el formato:
```
IP:PUERTO
```
Ejemplo:
```
192.168.1.100:16261
```
5. Guarda el servidor y conéctate.

## Problemas comunes

- **El servidor no aparece**: verifica que esté encendido en el panel de RDSNode.
- **Error de versión**: asegúrate de usar la misma versión del servidor.
- **Mods faltantes**: Project Zomboid descargará automáticamente los mods requeridos.
- **Puerto incorrecto**: confirma el puerto configurado en el servidor.

Con estos pasos ya podrás unirte correctamente a tu servidor de **Project Zomboid** alojado en **RDSNode** y comenzar tu aventura de supervivencia.
