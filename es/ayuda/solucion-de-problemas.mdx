# Solución de Problemas del Servidor

¿Tienes problemas? Empieza aquí.

## Lista Rápida de Verificación

- [ ] ¿El servidor aparece como "En línea" en el panel?
- [ ] ¿Estás usando la dirección IP correcta del servidor?
- [ ] ¿El cliente del juego está actualizado?
- [ ] ¿El servidor se reinició recientemente?
- [ ] ¿Hay errores en la consola?

## Problemas de Conexión

### No Puedo Conectarme al Servidor

1. **Revisa el estado del servidor** — Asegúrate de que esté en línea en tu panel  
2. **Verifica la IP y el puerto** — Copia la IP exacta desde tu panel  
3. **Revisa la versión del juego** — Asegúrate de que tu cliente coincida con la versión del servidor  
4. **Intenta reiniciar** — Detén y vuelve a iniciar el servidor  
5. **Revisa el firewall** — Desactívalo temporalmente para comprobar si está bloqueando  

### El Servidor Aparece Offline

1. **Reinicio forzado** — Detén el servidor, espera 30 segundos y vuelve a iniciarlo  
2. **Revisa los recursos** — Monitorea el uso de RAM/CPU, actualiza el plan si es necesario  
3. **Revisa la consola** — Busca mensajes de error o reportes de caída  

## Problemas de Rendimiento

### Lag en el Servidor

1. **Revisa los recursos** — Monitorea la RAM (debería mantenerse bajo el 80%) y el uso de CPU
2. **Desactiva plugins/mods innecesarios** — Elimina o actualiza los que estén obsoletos
3. **Optimiza el mundo** — Limpia chunks no utilizados, limita la aparición de entidades
4. **Ajusta la configuración** — Reduce la distancia de visión, optimiza el tick rate
5. **Actualiza el plan** — Si los recursos se saturan de forma constante

### Alto Uso de Memoria

1. **Revisa el uso de RAM** — Monitorea desde tu panel  
2. **Reinicia regularmente** — Ayuda a limpiar fugas de memoria  
3. **Actualiza plugins/mods** — Versiones antiguas pueden causar problemas  
4. **Aumenta la RAM** — Si te quedas sin memoria con frecuencia  

## Caídas del Servidor

### Caídas Frecuentes

1. **Revisa los logs de caída** — Busca mensajes de error en la consola  
2. **Actualiza todo** — Software del servidor, plugins/mods, versión de Java  
3. **Revisa conflictos** — Desactiva plugins/mods uno por uno para encontrar el problema  
4. **Revisa los recursos** — ¿Sin memoria? Actualiza tu plan  

### Errores al Iniciar

1. **Revisa la configuración** — Verifica server.properties y los permisos de archivos  
2. **Restaura desde una copia de seguridad** — Si los archivos están corruptos  
3. **Revisa dependencias** — Asegúrate de que la versión de Java sea compatible  
4. **Actualiza el software del servidor** — Usa la versión más reciente
