# Servidor Atascado en Pendiente

¿Tu servidor está atascado en “Pendiente”? Prueba estos pasos:

1. **Espera 10 minutos** — La mayoría de los servidores se activan automáticamente  
2. **Revisa tu correo electrónico** — Busca el correo “Account create” con tu enlace de inicio de sesión  
3. **Actualiza la página** — El estado podría haberse actualizado  
4. **Contacta con soporte** — Si sigue pendiente después de 10 minutos, [lo activaremos por ti](https://discord.gg/cMRsJZvp8X)

## ¿Sigues Teniendo Problemas?

**¿El pago fue cobrado pero el servidor no fue creado?**
- Tu servidor se creará una vez que el problema sea solucionado  
- No se realizarán cargos adicionales

**¿No puedes acceder a tu panel?**
- Espera primero el correo “Account create”  
- Revisa tu carpeta de spam  
- Usa el enlace de inicio de sesión del correo
